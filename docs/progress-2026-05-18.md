## 운영 정책

- 사용자가 ChatGPT, Gemini, YouTube Studio 같은 로그인 기반 브라우저 작업을 요청하면 Codex in-app browser를 우선 사용한다.
- 렌더링 결과물과 OAuth 토큰, `.env`, 브라우저 프로필, 다운로드 산출물은 커밋하지 않는다.
- YouTube 업로드는 사용자가 공개를 명시하지 않는 한 기본적으로 비공개로 올린다.
- 업로드 전 영상은 최소한 아래를 확인한다.
  - 자막이 겹치지 않는지
  - 썸네일 문구가 모바일에서도 읽히는지
  - 씬 내부 같은 이미지 구간에서 끊김이 생기지 않는지
  - 투자 권유나 수익 보장처럼 보이는 표현이 없는지
- 금융 콘텐츠에는 설명란 또는 영상 내 문구로 투자 유의 문구를 포함한다.
- 조회수 최적화 문구는 과장 수익 표현이 아니라 초보자 관점의 질문, 함정, 체크포인트 중심으로 작성한다.
